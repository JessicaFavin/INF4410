Pour lancer le TP,

Dans le code du client, vous devez changer le variable IP afin de vous connecter à votre serveur.

Ensuite, lancer le registre RMI dans le dossir bin puis compiler avec ant et pour lancer le serveur avec la commande ./server sur le serveur.

Dans le client, vous pouvez utiliser toute les commandes spécifiées dans le sujet du TP.
